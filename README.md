# Gym Website

A simple and responsive **single-page gym website** built using **HTML5, CSS3, and JavaScript**.

## Features
- Home page with hero section
- Facilities section
- Services section
- Membership plans section
- Contact section
- Dark mode toggle
- Responsive design for mobile and desktop

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
gym-website/
│── index.html
│── style.css
│── script.js
│── README.md

## How to Run
1. Download or clone the repository
2. Open the project folder in VS Code
3. Open `index.html` in browser or use Live Server

## Output
The website includes:
- Attractive homepage banner
- Gym facilities cards
- Services offered
- Membership pricing plans
- Contact details with interaction using JavaScript
